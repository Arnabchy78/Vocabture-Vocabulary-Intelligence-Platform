# Vocabture 🚀
### Vocabulary Intelligence Platform

Vocabture is an interactive web application designed to help users build and master their vocabulary through intelligent platform features, flashcards, and gaming modes.

## 🛠️ Tech Stack
* **Frontend:** React, TypeScript, Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Hosting:** Vercel

## 🚀 Live Demo
Check out the live deployment here: [Vocabture Live App](https://vocabture-vocabulary-intelligence-p-sand.vercel.app)
